## 简介
Mongoose是在node.js异步环境下对mongodb进行便捷操作的对象模型工具。<br />
那么要使用它，首先你得装上node.js和mongodb,关于mongodb的安装和操作介绍可以[参考](http://www.runoob.com/mongodb/mongodb-window-install.html)<br />
## Mongoose安装
```
npm install mongoose
```
